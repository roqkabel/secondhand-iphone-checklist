Step 5: functional checking


Plug the iPhone into an external battery/ AC sector.
If you hear a sound and the battery start charging, the test is a success

Record yourself via the Voice Memo app to test the mic and speaker

If you hear yourself and the sound is clean, the test is a success

Open any YouTube video from Safari and put max Volume to test the speakers
If you hear a clean sound, the test is a success

Switch the Mute button on the side and check if the notification Silent Mode On/Off appears

If the notification appears, the test is a success

Test the Volume button up & down and check if the volume notification silders go up & down
If the sliders are responsive, the test is a success

Test the Front Camera by taking a picture and view it in the Photos app

If the picture is clean, the test is a success

Test the Back Camera by taking a picture with each Zoom option and view it in the Photos app
If the picture is clean, the test is a success Each zoom will test a lens, so please take a picture with each of them

Record a video of 2 min with the Back Camera and view it in the Photos app
  • If the iPhone didn't shut down, the test is a success
  • If the video is clean, the test is a success
    •

    Connect any device via Bluetooth to test the connectivity
    If the device is paired and functional, the test is a success
